# 🟠 Bézier Inspector  
![Overview](overview.png)
Bézier Inspector is an Adobe Illustrator CEP extension for inspecting Bézier geometry directly on the artboard.  
Bézier Inspector는 Adobe Illustrator에서 Bézier 구조를 아트보드 위에 직접 표시해주는 CEP 확장입니다.

## ✨ Features / 기능

- Anchor points / 앵커 포인트
- Bézier handles / Bézier 핸들
- Handle connector lines / 핸들 연결선
- Baseline, x-height, cap height, ascent, descent guides / 베이스라인, x-height, cap height, ascent, descent 가이드
- Live overlay layer / 라이브 오버레이 레이어
